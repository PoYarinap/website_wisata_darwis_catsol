# 🌿 Website Wisata Kolam Ikan & Alam

Selamat datang di **Website Wisata Kolam Ikan & Alam** 🐟🌳
Sebuah website sederhana untuk memperkenalkan tempat wisata alam dengan suasana kolam ikan yang asri dan menenangkan.

---

## ✨ Deskripsi

Website ini dibuat untuk:

* Menampilkan informasi wisata kolam ikan dan alam
* Memberikan pengalaman visual yang menarik bagi pengunjung
* Memudahkan pengunjung untuk menghubungi admin melalui WhatsApp

---

## 🚀 Fitur Utama

* 🌄 Tampilan halaman wisata yang menarik
* 🖼️ Galeri gambar (kolam ikan & alam)
* 📱 Responsive (mobile & desktop)
* 💬 Form kontak langsung ke WhatsApp
* ⚡ Interaksi cepat tanpa backend/database

---

## 💬 Sistem Form WhatsApp

Pengunjung dapat mengisi form seperti:

* Nama
* Pesan

Setelah dikirim:

* Data otomatis diarahkan ke WhatsApp admin
* Pesan langsung terbuka di WhatsApp
* Tidak menggunakan database

### Contoh Alur:

1. User isi form
2. Klik kirim
3. Redirect ke WhatsApp
4. Pesan sudah otomatis terisi
